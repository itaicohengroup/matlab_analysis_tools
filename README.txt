This directory includes:
1. "barnesn" : N-dimensional Barnes interpolation (for interpolating scattered data)
2. "MTImport": Function to import images collected using the Zeiss 'MultiTime' macro on Cohen group confocal microscopes
3. "ncorr_tools": (a) Modified version of NCorr that saves displacement gradient data and (b) Custom plotting function "ncorr_plot.m" for plotting NCorr data (see http://www.ncorr.com/)
4. "rearrange_tilescan": A function to re-arrange the tiles in an image (i.e. to fix the mis-arranged tiles produced by Cohen group confocals)
