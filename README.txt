This directory includes:
1. "barnen" : N-dimensional Barnes interpolation (for interpolating scattered data)
2. "MTImport": Function to import images collected using the 'MultiTime' macro on our confocals
3. "ncorr_tools": (a) Modified version of NCorr that saves displacement gradient data and (b) Custom plotting function "ncorr_plot.m" for plotting NCorr data
4. "rearrange_tilescan": A function to re-arrange the tiles in an image (i.e. to fix the mis-arranged tiles from our confocals)
